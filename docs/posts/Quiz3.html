문제 1. 미분 – 10점

함수 \(f(x)=\cos(x)e^{-0.1x}\) 를 입력하면 도함수를 리턴하는 함수를 파이썬으로 구현하라. 이러한 함수를 이용하여 \(f(x)\), \(f'(x)\), \(f''(x)\)의 그래프를 한 Figure에 모두 그려라. (범례는 표현하지 않아도 감점없음)

  • \(x\)의 범위는 \((-2,15)\) 로 할 것.
  • tip: 욕심부려서 \(h\)의 값을 너무 작게 설정하면 그림이 예쁘게 안나옵니다.

(풀이)

def f(x):
    return np.cos(x)*np.exp(-0.1*x)
x = np.linspace(-2,15,1000)
def d(f):
    return lambda x: (f(x)-f(x+0.01))/0.01
f1 = d(f)
f2 = d(f1)
f3 = d(f2)
plt.plot(x,f(x))
plt.plot(x,f1(x))
plt.plot(x,f2(x))
plt.plot(x,f3(x))

문제 2. 적분 – 10점

(1) 다음 적분값을 근사계산하라. – 5점

\[\int_{0}^{\pi} e^{\sin(x)} \, dx\]

x = np.linspace(0,np.pi,1000)

(풀이)

x = np.linspace(0,1,1000)
(np.log(1+x)/(1+x**2)).mean() * 1
np.float64(0.27209925834918985)

문제 4. lambda와 map – 20점

각 소문항 4점씩

(1) map()과 lambda를 이용하여 리스트 [1, 2, 3, 4, 5]의 각 원소를 제곱한 리스트를 만들어라.

## 정답은 아래와 같음
[1, 4, 9, 16, 25]

(풀이)

list(map(lambda x: x**2,[1,2,3,4,5]))
[1, 4, 9, 16, 25]

(2) 두 개의 리스트 x = [1, 2, 3, 4, 5]와 y = [10, 20, 30, 40, 50]가 주어졌을 때, map()과 lambda를 이용하여 각 원소의 합을 구하라.

## 정답은 아래와 같음
[11, 22, 33, 44, 55]

(풀이)

list(map(lambda x,y: x+y, [1,2,3,4,5], [10,20,30,40,5]))
[11, 22, 33, 44, 10]

(3) map()과 lambda를 이용하여 리스트 [1, 2, 3, 4, 5]의 각 원소에 대해 \(a_n = 2n + 3\)의 값을 계산하라.

## 정답은 아래와 같음
[5, 7, 9, 11, 13]

(풀이)

list(map(lambda n: 2*n + 3, [1,2,3,4,5]))
[5, 7, 9, 11, 13]

(4) map()과 lambda를 이용하여 리스트 [1, 2, 3, 4, 5, 6]의 각 원소에 대해 \(a_n = n^3 - n\)의 값을 계산하라.

## 정답은 아래와 같음
[0, 6, 24, 60, 120, 210]

(풀이)

list(map(lambda n: n**3 - n, [1,2,3,4,5,6]))
[0, 6, 24, 60, 120, 210]

(5) map()과 lambda를 이용하여 리스트 ['Python', 'Java', 'C++', 'JavaScript']의 각 문자열을 대문자로 변환하라.

## 정답은 아래와 같음
['PYTHON', 'JAVA', 'C++', 'JAVASCRIPT']

(풀이)

list(map(lambda s: s.upper(), ['Python', 'Java', 'C++', 'JavaScript']))

문제 5. 순열과 중복제거 – 10점

6개의 문자 \(a, a, b, b, c, d\)를 모두 일렬로 나열하는 경우의 수를 파이썬을 이용하여 구하여라.

  • itertools.permutations()을 이용할 것.

hint: 답은 \(\frac{6!}{2!2!}=180\) 이다.

(풀이)

len(set(itertools.permutations('aabbcd')))
180

문제 6. – 30점

(1) 함수 \(g(x) = \cos x\)라 할 때, \(1 \leq x \leq 10\)에서 부등식 – 10점

\[g(x) > \frac{\sqrt{3}}{2}\]

을 만족시키는 모든 자연수 \(x\)의 개수를 구하시오.

(풀이)

x = np.array(list(range(1,11)))
g = lambda z: np.cos(z)
(g(x) > np.sqrt(3) / 2 ).sum()
np.int64(1)

(2) 함수 \(h(x) = e^{-0.5x}\)라 할 때, \(1 \leq x \leq 10\)에서 부등식 – 10점

\[h(x) \cdot h(x+1) < 0.001\]

을 만족시키는 모든 자연수 \(x\)의 값의 곱을 구하시오.

(풀이)

h = lambda z: np.exp(-0.5*z)
(x[h(x) * h(x+1) < 0.001]).prod()
np.int64(5040)
